<h1 align="center"> Forex Sentiment Analysis Application</h1>


<p align="center">
  A Python-based <b>Forex market sentiment analysis</b> app that classifies financial news as <b>positive</b>, <b>neutral</b>, or <b>negative</b>.
</p>


---

<h2> Features</h2>

<ul>
  <li> Input Forex News Text</li>
  <li> Analyze Forex market sentiment in real-time</li>
  <li> Clean, intuitive web interface powered by Streamlit</li>
  <li> Classify financial news as Positive, Neutral, or Negative</li>
  <li> View Clear Results & Confidence</li>
</ul>

<h2> Installation</h2>

<pre>
# Clone repository
git clone https://github.com/ramiaffes/Forex-Sentiment-Analysis-Application.git
cd Forex-Sentiment-Analysis-Application

# Create virtual environment
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
</pre>

---

<h2> Running the App</h2>

<pre>
streamlit run app.py
</pre>

<p>Open your browser and go to <code>http://localhost:8501</code> to interact with the app.</p>
